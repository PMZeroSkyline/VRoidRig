# VRoidRig
Blender VRoid Rig Addon

1. Import vrm model using https://github.com/saturday06/VRM-Addon-for-Blender addon.
![](doc/import_vrm.png)
2. Select Armature click Rig button in VRoidRig view 3d panel.
![](doc/rig.png)
3. Done!
![](doc/done.png)
